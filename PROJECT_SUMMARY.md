## 如何启动项目

### 第一步：安装依赖
```bash
npm install
```

### 第二步：配置API密钥
```bash
# 复制环境变量示例
cp .env.local.example .env.local

# 编辑 .env.local，添加你的DeepSeek API密钥
DEEPSEEK_API_KEY=your_api_key_here
```

### 第三步：启动开发服务器
```bash
npm run dev
```

### 第四步：访问应用
打开浏览器访问：http://localhost:3000
